使用git的阶段性总结

Date Mon 20 October 2014 Tags git

一般使用者

在某分支中干活

git add -> git commit -> git pull -> git push

分支间穿梭

git stash 来方便的保留现场,然后跑到别的分支处理一些紧急事务

git checkout checkout到分支、commit、当前工作目录下某一文件的某一个历史状态，总之是个git内部存在的hash值就行.

git branch也许需要创建个本地分支来玩耍,然后删掉
